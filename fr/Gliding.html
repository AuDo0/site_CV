Vol en planeur

Programme d'analyse d'un vol

C'est un code basé sur les bibliothèques Python opensoar et PySoar permettant d'analyser une trace de vol.

Statistiques calculés :

Ce code est une aide pour comprendre ses points forts et ses points faibles pour que chacun puissent s'améliorer. Le code calcule pour chaque ascendance et transition des statisques comme:

Statistiques en ascendance Statistiques en transition
Durée Durée
Finesse moyenne (Ave LD) Sens de spirale (left / right / mixed)
Distance Vario moyen (Ave vario)
Vitesse de vol moyenne (Ave speed) Gain d'altitude (Alt gain)

Le résultat est affichée sur une carte.
