Gliding Analyse Program

Python code based on opensoar and PySoar package.

Code ability :

This code aims to analyze a gliding flight path to improve himself. The code compute the following statistic for each thermal and cruise :

Cruise statistic Thermal statistic
Duration Duration
Average LD (Ave LD) Spiral direction (left / right / mixed)
Distance Average vario (Ave vario)
Average speed (Ave speed) Altitude gain (Alt gain)

The result is plot on a map.
